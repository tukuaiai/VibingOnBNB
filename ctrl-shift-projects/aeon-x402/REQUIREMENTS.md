# AEON x402 付费 Agent - 需求说明

## 来源映射
- 任务规划：`$PROJECT_ROOT/new/ctrl-shift-challenges/task-aeon-x402.md`

## 目标
- 在 24 小时内完成“AI Agent 能自动支付并调用付费 API”的可演示闭环。

## 功能需求
1) 付费 API 支持 HTTP 402 挑战流程  
2) 客户端/Agent 自动处理 402 -> 支付 -> 重试  
3) 记录支付凭证与回执（日志即可）  
4) 固定价格/按路由计费配置  
5) 最小演示脚本或页面（CLI 优先）

## 非目标
- 多链兼容与复杂结算策略
- 动态定价与风控体系
- 生产级高可用与安全审计

## 依赖与可复用轮子
- AEON bnb-x402 示例仓库
- x402 协议文档
- openlibx402（如需更快服务端集成）
- viem/ethers（用于钱包签名）

## 验收标准
- 首次请求返回 402，客户端自动支付后重试成功返回 200
- 日志中可见支付回执或交易哈希
- 付费 API 可被 Agent 调用并完成支付

## 约束与假设
- 以 BNB Chain 测试网为首选环境
- 支付金额固定，避免复杂计价
